PID 控制器的计算函数

参数
ref_ls参考值列表，只使用第一个值，如果为nullptr则使用 0
fdb_ls反馈值列表，只使用第一个值，如果为nullptr则使用 0
ffd_ls前馈值列表，只使用第一个值，如果为nullptr则使用 0
out_ls输出值列表，只使用第一个值，为 nullptr 时，输出传递失败
返回
表示计算过程是否正常的状态值
返回值
kStateOk计算成功且所有参数均不为nullptr
kStateError有参数为 nullptr，为了计算成功会使用默认值
注意
- 当 ref_ls 为 nullptr 时，使用 0 作为参考值
- 当 fdb_ls 为 nullptr 时，使用 0 作为反馈值
- 当 ffd_ls 为 nullptr 时，使用 0 作为前馈值
- 当 out_ls 为 nullptr 时，输出传递失败
