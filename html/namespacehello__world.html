◆ PeriodDataSub()

float hello_world::PeriodDataSub ( float  minuend,
float  subtrahend,
float  period 
)
inline

处理角度越过0值的情况，以半周期为基准

参数
ref指向目标角度的指针
fdb实际反馈的角度值
period角度值的周期，通常是180度

处理角度越过0值的情况，确保目标角度和实际反馈的角度值在同一半周期内。 如果目标角度 - 实际反馈的角度 >= half_period，则减去2 * half_period，确保在同一半周期内。 如果目标角度 - 实际反馈的角度 < -half_period，则加上2 * half_period，确保在同一半周期内。
