)

构造函数，初始化 MultiNodesPid 对象，至少会存在一个 PID 控制器节点，推荐和 paramsAt 函数一起使用

参数
typePID 类型
out_limit输出限制
countPID 数量

如果 count 为 0，则将其设置为 1。 然后，根据 count 的值创建相应数量的 PID 控制器，并将它们添加到 pid_list_ 中。 反馈值维数为 PID 控制器节点的数量。 如果 type 为 kParallel，则将参考值维数与反馈值维数相同。 前馈值维数和输出值维数始终为 1。

注解
如果需要修改 PID 参数，推荐使用 paramsAt 函数
相较于初始化时确定参数更加安全、灵活，不必预先声明参数数组

计算级联 PID 控制器的输出

参数
ref_ls参考值数组，只使用第一个值，如果为nullptr则使用 0
fdb_ls反馈值数组，将使用等同于 size() 个数的值，如果为 nullptr 将不会进行计算，且会将输出置零
ffd_ls前馈值数组，只使用第一个值，如果为 nullptr 则使用 0
out_ls输出值数组，只使用第一个值，为 nullptr 时输出传递失败
返回
表示计算过程是否正常的状态值
返回值
kStateOk计算成功且所有参数均不为 nullptr
kStateError有参数为 nullptr，计算成功与否视参数而定
注意
- 当 ref_ls 为 nullptr 时，使用 0 作为参考值
- 当 fdb_ls 为 nullptr 时，计算失败，并将输出置零
- 当 ffd_ls 为 nullptr 时，使用 0 作为前馈值
- 当 out_ls 为 nullptr 时，输出传递失败
